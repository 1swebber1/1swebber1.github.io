    Looping in Ruby

    What Technically is a Loop?

    When a computer program executes an action it more than likely has to perform said action a number of times, not just once, until a certain condition is met. Therefore a loop simply repeats the action until a condition is met. Ruby has a variety of loops, which consist of ‘while’, ‘until’, ‘each’ and ‘times’ loops with the more popular being ‘while’ and ‘until’ and the more favored being ‘each’.

    While Loops

    A while loop will execute the code so long as the condition that is met is true. It will run until a particular expression is equal to false or until the break command is inserted. While loops are excellent for if you want to execute code until a particular value is reached.
